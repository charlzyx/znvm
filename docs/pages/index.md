# znvm

**znvm** — A blazingly fast Node.js version manager built with Zig. The ultimate **nvm alternative** that solves the **nvm-slow** problem.

[Get Started](/guide) [GitHub ↗](https://github.com/charlzyx/znvm)

## Why znvm?

Tired of slow **node version manager** tools? **znvm** is designed for developers who demand speed:

- **⚡️ Instant Performance**: Starts in < 5ms (vs nvm's 150ms+ or fnm's 20ms)
- **🎯 Unix-First**: Lean and focused, no cross-platform bloat
- **🛠 Zero Configuration**: Works with your existing `.nvmrc`

## The Problem with Existing Tools

| Manager | Language | Startup | Issues |
| :--- | :--- | :--- | :--- |
| `nvm` | Bash | ~150ms+ | **nvm-slow**, shell lag |
| `fnm` | Rust | ~20ms | Feature bloat |
| **`znvm`** | **Zig** | **< 5ms** | Fast & minimal |
